# Portfolio Management Optimization with Time Series Forecasting

## Overview

Guide Me in Finance (GMF) Investments is a forward-thinking financial advisory firm specializing in personalized portfolio management. This project applies **time series forecasting** techniques to historical financial data to optimize asset allocation, predict market trends, and enhance portfolio performance.

By combining classical statistical models (ARIMA/SARIMA) and deep learning models (LSTM), this project aims to:

- Forecast Tesla's stock price trends.
- Analyze risk metrics for a multi-asset portfolio.
- Recommend optimized portfolio allocations using Modern Portfolio Theory (MPT).
